两数之和

给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 target 的那 两个 整数，并返回它们的数组下标。

你可以假设每种输入只会对应一个答案，并且你不能使用两次相同的元素。

你可以按任意顺序返回答案。

示例 1：

输入：nums = [2,7,11,15], target = 9
输出：[0,1]
解释：因为 nums[0] + nums[1] == 9 ，返回 [0, 1] 。

示例 2：

输入：nums = [3,2,4], target = 6
输出：[1,2]

示例 3：

输入：nums = [3,3], target = 6
输出：[0,1]

提示：

  • 2 <= nums.length <= 104
  • -109 <= nums[i] <= 109
  • -109 <= target <= 109
  • 只会存在一个有效答案

遍历：

#include <vector>
using namespace std;

class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        int i;
        int j;
        int len = nums.size();
        for(i = 0; i < len; i++){
            for(j = i + 1; j < len; j++){
                if(nums[i] + nums[j] == target)
                    return {i, j};
            }
        }
        return {};
    }
};

时间复杂度：O(n²)

两层循环嵌套

空间复杂度：O(1)

只使用到三个临时变量，空间复杂度低

哈希表:

class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        unordered_map<int,int> hashtable;
        //定义哈希表
        for(int i=0;i<nums.size();i++){
            auto item = hashtable.find(target-nums[i]);
            if(item != hashtable.end())
                return {item->second,i};
            hashtable[nums[i]]=i;
        }
    return {};
    }
};

时间复杂度：O(n)

循环只走一遍数组，一共 n 次循环

时间复杂度：O(n)

最差情况下（找不到配对，要把所有数字都存入哈希表），哈希表会存 n 组键值对，占用额外空间随 n 线性增长

知识点

map & unordered_map

unordered_map

unordered_map 不保证元素的排序，但能提供更快的查找速度

  1. 定义
#include<unordered_map>;
unordered_map<int,int> hashtable; //定义一个 <int,int> 类型的 unordered_map
  1. 查找
int need = 2;
auto item = hashtable.find(need);

if (item != hashtable.end())
    {
        cout << "key：" << item->first << endl;
        cout << "value：" << item->second << endl;
    }
  1. 赋值
hashtable[5] = 1; // first = 5，second = 1
hashtable[9] = 1; // first = 9，second = 1
  1. 插入
hashtable.insert({11, 2}); //插入 11 2
  1. 插入不会覆盖已有键
unordered_map<int, string> mp;
// insert特点：如果key已经存在，不会覆盖原来的值
mp.insert({1, "第一个人"});
mp.insert({2, "第二个人"});
// 尝试再次插入key=1，不会修改原有value
mp.insert({1, "我想覆盖前面的数据"});
for (auto& item : mp)
{
    cout << item.first << " -> " << item.second << endl;

}
// 1 -> 第一个人
// 2 -> 第二个人
  1. 遍历
for (auto &item : hashtable)
{
    cout << item.first << " " << item.second << endl;
}
  1. .empty()
if (hashtable.empty()) // 返回 false 或者 true
  1. .erase()
hashtable.erase(5); // 删除 key 为 5 的一组数据
  1. .size()
int len = hashtable.size(); //返回容器存储的键值对数量
  1. .clear()
hashtable.clear(); // 清空哈希表所有内容

map

特性：底层红黑树，自动按 key 升序排序，增删查找 O (logn)，速度慢于 unordered_map

查找、赋值、insert 插入、遍历、所有成员函数（empty/erase/size/clear）基础语法完全和 unordered_map 一致

区别：

① 遍历容器时，输出永远按 key 从小到大有序排列；


map<int, int> mp;
 // 乱序插入key
mp[5] = 100;
mp[2] = 200;
mp[8] = 800;
// 遍历输出，自动从小到大排好key
for (auto& item : mp)
{
    cout << item.first << " " << item.second << endl;
}
//输出结果
//2 200
//5 100
//8 800

② 运行效率低于 unordered_map，适合需要有序输出、范围查找的场景

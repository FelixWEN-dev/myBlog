字母异位词分组

给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。

 

示例 1:

输入: strs = [“eat”, “tea”, “tan”, “ate”, “nat”, “bat”]

**输出: **[[“bat”],[“nat”,”tan”],[“ate”,”eat”,”tea”]]

解释：

  • 在 strs 中没有字符串可以通过重新排列来形成 "bat"。
  • 字符串 "nat" 和 "tan" 是字母异位词，因为它们可以重新排列以形成彼此。
  • 字符串 "ate" ，"eat" 和 "tea" 是字母异位词，因为它们可以重新排列以形成彼此。

示例 2:

输入: strs = [“”]

**输出: **[[“”]]

示例 3:

输入: strs = [“a”]

**输出: **[[“a”]]

 

提示：

  • 1 <= strs.length <= 104
  • 0 <= strs[i].length <= 100
  • strs[i] 仅包含小写字母

哈希表

思路：

  1. 创建哈希表
  2. 遍历数组
  3. 对数组中每个字符串进行排序
  4. 将排序后的字符串储存进哈希表中
  5. 创建动态数组
  6. 遍历哈希表
  7. 输出结果

题解：

class Solution {
public:
    vector<vector<string>> groupAnagrams(vector<string>& strs) {
        unordered_map<string, vector<string>> mp; //创建一个哈希表
        //遍历数组
        for(int i = 0; i < strs.size(); i++){
            string key = strs[i]; //将strs[i] 赋值给临时变量 key
            sort(key.begin(), key.end());  //对字符串进行排序
            mp[key].push_back(strs[i]);
            //将排序后的字符串储存进哈希表中
            //去哈希表里找键等于 key 的那一组
            //如果不存在这个 key，会自动新建一个空的 vector<string> 绑定给这个 key
            //哈希表中的结构：[key , vector<string>]
        }

        vector<vector<string>> ans;
        for(auto item = mp.begin(); item != mp.end(); item++){
            ans.push_back(item->second);
        }

        return ans;
    }
};

总结

字母异位词分组核心思想

由相同字母、相同个数组成，仅字母排列顺序不同的字符串互为字母异位词；
对字符串内部字符排序后，所有异位词会得到完全一致的字符串，可作为统一标识用来分组。

  1. 创建哈希表
  2. 遍历数组
  3. 对数组中每个字符串进行排序
  4. 将排序后的字符串储存进哈希表中
  5. 创建动态数组
  6. 遍历哈希表
  7. 输出结果

复杂度分析

时间复杂度：(O(n \cdot k\log k))
每个字符串排序耗时 (O(k\log k))，共遍历 n 个字符串；哈希插入、遍历操作均为线性低阶消耗，不主导复杂度。
空间复杂度：(O(nk))
哈希表存储全部 n 个字符串的所有字符，额外无大规模临时空间开销。
